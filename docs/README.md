# Better-Stack Docs

This is the documentation app for better-blog in the monorepo.

## Development

```bash
pnpm -w install
pnpm -F "@btst/*" build   # ensure the library is built once
pnpm --filter better-stack-docs dev
```

- The docs import from the local package `better-stack` via `workspace:*`.
- `next.config.mjs` sets `basePath: '/docs'` and `transpilePackages: ['better-stack']`.
- Visit `http://localhost:3000/docs`.

## Build

```bash
pnpm -F "@btst/*" build
pnpm --filter better-stack-docs build
```

The static output is generated by Next.js in `.next/`. Deploy with Vercel or any Next-compatible host.

## Type tables (fumadocs)

MDX files reference source files for API tables, e.g.:

```mdx
<AutoTypeTable path="../../packages/better-stack/src/api/types.ts" name="BlogApiRouterOptions" />
```

Ensure the relative paths stay correct if you move files.
