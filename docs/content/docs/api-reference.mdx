## Context (`@btst/stack/context`)

The context module exports React hooks and components for managing plugin configuration.

**Exported Functions:**
- `BetterStackProvider` - Context provider component
- `useBetterStack()` - Hook to access the full context
- `usePluginOverrides()` - Hook to access plugin-specific overrides  
- `useBasePath()` - Hook to access the base path
